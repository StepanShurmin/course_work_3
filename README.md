IT-отдел крупного банка делает новую фичу для личного кабинета клиента.
Это виджет, который показывает несколько последних успешных банковских операций клиента. 
Вам доверили реализовать алгоритм, который на бэкенде будет готовить данные для отображения в новом виджете.

## Задача

Реализовать функцию, которая выводит на экран список из 5 последних выполненных клиентом операций в формате:

<дата перевода> <описание перевода>
<откуда> -> <куда>
<сумма перевода> <валюта>
